= Chart -- A chart update via comet

== Running the demo

* Download the Dojo Toolkit: Dojo + Dijit + DojoX at http://www.dojotoolkit.org/downloads and copy all files in static/dojo
* run chart.rb
* Open some browsers at http://localhost:3000

If you have the exception RCometAddrInUse change the port 8990 in chart.rb and in chart.html
